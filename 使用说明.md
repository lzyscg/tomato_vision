# 番茄 IP 数据看板使用说明

## 最简单的使用方式

双击项目目录里的：

```text
番茄看板.html
```

即可直接打开看板。这个文件是已导出的数据库快照，不需要启动服务。

如果需要更新到最新数据库数据，请让配置过数据库的电脑重新导出一次 HTML。

## 实时查询版怎么启动

### Windows 生产人员推荐

直接双击：

```text
dist/番茄IP数据看板-2.0.0-x64.exe
```

这个版本已经内置运行环境和数据库配置，不需要安装 Node.js。

### macOS

双击项目目录里的：

```text
启动番茄看板.command
```

### Windows

右键项目目录里的：

```text
启动番茄看板.ps1
```

选择：

```text
使用 PowerShell 运行
```

如果系统不允许直接运行脚本，可以打开 PowerShell 后执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\启动番茄看板.ps1
```

## 环境要求

双击 `番茄看板.html` 不需要安装环境。

脚本方式启动实时查询版、重新导出 HTML、重新打包 exe 需要电脑里安装 Node.js 18 或更高版本。已经打包好的 Windows exe 不需要安装环境。

第一次启动会自动安装项目依赖，可能需要几十秒到几分钟。以后再启动会快很多。

## 数据库配置

脚本方式和重新打包需要项目根目录下有 `.env.local` 文件，内容格式如下：

```text
PGHOST=10.128.1.3
PGPORT=5432
PGDATABASE=qiyin_warehouse
PGUSER=dbuser_view
PGPASSWORD=数据库密码
PGSSL=prefer
```

如果第一次启动时没有 `.env.local`，程序会自动创建模板并打开，请补全数据库密码后重新启动。

## 使用看板

打开 `番茄看板.html` 后可以直接浏览导出时刻的数据：

- 顶部可以选择日期范围
- 第一层展示歌手粉丝趋势
- 可以搜索歌手或按风格筛选
- 点击歌手后进入第二层，查看该歌手歌曲收藏趋势
- 点击右上角「刷新页面数据」会按当前筛选条件重新筛选本地快照

实时查询版打开后会直接查询 PostgreSQL 数据库。

## 停止服务

页面右上角有：

```text
关闭服务
```

测试结束后点击它即可停止本地服务。
